Employee Manager Android Application

This project is built for android and it has two applications.
1. EmployeeAdmin
2. EmployeeUser

EmployeeAdmin

This application has the following functionalities:
1. Upload/Delete Notice
2. Upload Ebook
3. Update/Delete Employee

EmployeeUser

This application has the following functionalities
1. View Notice
2. View Other Employees
3. View All Ebooks
4. Read/Download PDFs inside the application.

Features of EmployeeAdmin

1. Upload Notice

Upload an image of notice and give it a title.
It will store the notice in FirebaseStorage to display the notices in EmployeeUser application.

2. Delete Notice

This activity has a button to delete a notice that was previously uploaded.
It will delete the notice from FirebaseStorage.

3. Upload Ebook

Upload a PDF with a title. That stores the PDF File to FirebaseStorage and its corresponding title.

4. Add Employee

On clicking the FloatingAction Button it opens an activity where we can insert a new record of the employee with his/her image.

5. Update / Delete Employee

On clicking "UPDATE INFO" button it opens a new activity where we can edit any field of record of an employee and click on "UPDATE" button to update the record.
We can also click on "DELETE" button to delete the record of that employee.

Features of EmployeeUser

1. View Notice

It will display all the notices with upload date/time uploaded till date to FirebaseStorage.
